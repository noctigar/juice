======================
商品一覧API
======================

内容
======
#. 商品の一覧を取得する。

リクエスト
-----------
/item

返ってくる値
-------------
- 商品名
- あったかい or つめたい
- 値段
- 売り切れフラグ

説明
======
- ユーザーが商品を選ぶ際の一覧

補足
----
- 売切商品も取得する（フラグで判別できます）。
- 業者側で登録を削除した商品は取得しない。
- 業者の在庫確認とは違い、登録されている商品は全て取得する。
- 在庫は取得しない。
